# Raddare

A simple route planner service.

## Docker
```shell
$ docker run --rm adrianpksw/raddare:stage
```

## Clone
```shell
$ git clone https://github.com/adrianpk/raddare.git
```

## Run
```shell
$ make run
```

## API Call
```shell
$ make curl-routes
```
